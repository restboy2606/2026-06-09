# 01. AICADEMY — 온라인 AI 동영상 교육 플랫폼 구축

> rest04 초기 구축 개발일지 · 2026-06-09

## 1. 목표

[rest03 건설사 템플릿](https://github.com/aebonlee/rest03)의 React + Vite + Tailwind 구조를 베이스로,
**AI / AI 리터러시 동영상 강의를 제공하는 교육회사 사이트**를 신규 구축한다.

### 요구사항
- 주컬러 **다크블루 + 로열블루**, 포인트·보조 포함 **5색 컬러 팔레트**
- **다크모드 & 라이트모드** 전환
- **모바일 최적화**
- 유튜브 **미등록(링크공개) 영상**을 임베드하여 노출
- 동영상을 **2×3 그리드**로 배치 + **페이지네이션**
- **주제별 메뉴**로 영상 구분

## 2. 베이스 분석 (rest03 → rest04)

rest03에서 가져온 검증된 구조:
- `HashRouter` 기반 SPA (GitHub Pages 정적 호스팅 호환)
- `data/site.js` 단일 데이터 소스 → 콘텐츠 수정 용이
- `Header`/`Footer`/`ScrollToTop` 등 공통 컴포넌트 패턴
- `Business.jsx`의 **카테고리 탭 + 그리드 + 페이지네이션** 패턴
  → 영상 강의 페이지(`Videos.jsx`)의 골격으로 재활용
- `.github/workflows/deploy.yml` Actions 자동배포 그대로 사용 (base만 `/rest04/`로 변경)

## 3. 컬러 팔레트 (5색)

`tailwind.config.js`에 시맨틱 토큰으로 정의, 각 색상은 50~950 음영 포함.

| 토큰 | 역할 | 대표 색 |
| --- | --- | --- |
| `navy` | 주컬러(다크블루) — 배경/타이포 | `#0B1A33` |
| `royal` | 주컬러(로열블루) — CTA/링크/강조 | `#2B5BFF` |
| `sky` | 보조컬러(하늘색) | `#38BDF8` |
| `amber` | 포인트컬러(골드앰버) | `#FFB020` |
| `mint` | 포인트컬러2(민트) | `#34D399` |

- 히어로 배경은 navy→royal 그라데이션 + sky/royal radial glow (`.hero-gradient`)
- `.surface`, `.btn-primary`, `.chip`, `.eyebrow` 등 공통 컴포넌트 클래스로 일관성 확보

## 4. 다크/라이트 모드

- Tailwind `darkMode: 'class'` 전략
- `src/theme/ThemeContext.jsx` — 전역 상태 + `localStorage('theme')` 저장 + OS 설정(`prefers-color-scheme`) 추종
- `index.html` 인라인 스크립트로 **페인트 전 테마 적용 → FOUC(깜빡임) 방지**
- 헤더의 `ThemeToggle`(🌙/☀️)로 즉시 전환

## 5. 영상 강의 페이지 (핵심)

`src/pages/Videos.jsx`
- URL 파라미터 `:topic`으로 주제 전환 (`/videos/ai-basics` 등)
- 상단 **sticky 주제 탭** (모바일에서 가로 스크롤)
- **2열 × 3행 = 페이지당 6개** 그리드 (`PER_PAGE = 6`)
- `Pagination` 컴포넌트 — 처음/이전/번호/다음/마지막, 페이지 변경 시 그리드 상단으로 스크롤

### 컴포넌트
- `VideoCard` — 16:9 썸네일(유튜브 `hqdefault`, 없으면 그라데이션 폴백) + 재생 오버레이 + 레벨 뱃지 + 재생시간
- `VideoModal` — 유튜브 `embed` iframe(autoplay/rel=0), ESC 닫기 + 배경 스크롤 잠금

### 주제 (topics)
`ai-basics`(AI 기초) · `ai-literacy`(AI 리터러시) · `generative`(생성형 AI) · `prompt`(프롬프트) · `ethics`(AI 윤리)
→ `topics` 배열만 수정하면 네비·탭·라우트·푸터에 자동 반영.

## 6. 페이지 구성

| 페이지 | 경로 | 내용 |
| --- | --- | --- |
| 홈 | `/` | 히어로 · 특징 4종 · 주제 카드 · 추천영상 · CTA |
| 영상 강의 | `/videos/:topic` | 주제탭 + 2×3 그리드 + 페이지네이션 |
| 회사소개 | `/about` | 소개문 · 강점 · 커리큘럼 요약 |
| 문의 | `/contact` | 연락처 + 문의 폼 |
| 약관/개인정보 | `/terms` `/privacy` | 단순 텍스트 |

## 7. 유튜브 연동 가이드

- 영상 공개범위는 **"미등록(링크 공개)"** 사용 (비공개는 임베드 재생 불가)
- `src/data/site.js` → `videos`의 각 강의 `youtubeId`(11자리)만 입력하면 썸네일·재생 자동 연결
- 현재는 데모 데이터(youtubeId 빈 값) → 그라데이션 플레이스홀더로 동작

## 8. 빌드 / 배포

- `npm run build` 성공 (49 modules, CSS 27.9KB / JS 196KB)
- `main` push → GitHub Actions가 빌드 후 GitHub Pages 배포
- 배포 URL: `https://aebonlee.github.io/rest04/`
