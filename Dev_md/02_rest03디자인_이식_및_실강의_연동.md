# 02. rest03 디자인 이식 + 실제 강의 영상 연동

> 2026-06-09 · 2차 개발일지

## 1. 배경 (피드백 반영)

1. **"rest03 디자인을 더 가깝게"** — 구조·패턴뿐 아니라 시각 언어까지 rest03에 맞춰달라.
2. **"다크모드만 떠서 답답"** — 라이트 모드를 기본으로.
3. **실제 커리큘럼/유튜브 링크 제공** — 받은 30개 영상을 모두 사이트에 연동.

## 2. rest03 디자인 이식

rest03(건설사 템플릿)의 시그니처 요소를 그대로 가져옴:

| 요소 | 적용 |
| --- | --- |
| 컨테이너 폭 | `max-w-container` 1280 → **1600px** |
| 좌우 여백 | `section-x` = `px-4 md:px-10 lg:px-24 xl:px-40` (rest03 동일) |
| 2단 푸터 | 상단 밝은 소개영역 + **하단 다크(navy-950) 바**(연락처·링크·정책) |
| SubPageLayout | 큰 섹션타이틀(navy-300) + 빵부스러기 + sticky 탭 + 페이지헤드 밴드 |
| 영상 페이지 | rest03 `Business.jsx`의 **대탭 + 소분류 버튼 + 그리드 + 페이지네이션** 패턴 이식 |

## 3. 다크/라이트

- 기본값 **라이트 모드** (OS 다크 자동추종 제거, 토글로 명시 선택 시에만 다크)
- 서브페이지 헤더는 어두운 네이비 → **밝은 `hero-soft` 그라데이션**(흰색·하늘빛)
- 홈 히어로만 랜딩 임팩트용 진한 네이비 유지

## 4. 콘텐츠 구조 (실제 강의 연동)

받은 30개 영상을 **2개 대주제 + 모듈(소분류)** 로 재구성:

### ① AI 콘텐츠 제작 (`/videos/creation`) — 17강
콘텐츠 제작(2) · 이미지 만들기(2) · 영상 제작(3) · 프레젠테이션(3) · 효과적 발표(1) · 피그마(2) · 바이브코딩(4)

### ② AI 리터러시 (`/videos/literacy`) — 13강
창의·문제해결(3) · 디자인씽킹(1) · 컴퓨팅 사고(1) · 마크다운(2) · 노션(2) · 리포트 작성(3) · 인용·출처(1)

- 영상 페이지: **대탭(2주제)** → **소분류 버튼(전체/모듈)** → **2×3 그리드** → **페이지네이션**
- 모듈 선택 시 해당 모듈 영상만 필터링, 카드 칩에 모듈명 표기
- 전 영상 **oEmbed로 임베드 가능 검증 완료**

## 5. 빌드/배포

- `npm run build` 성공 (50 modules)
- `main` push → GitHub Actions 자동배포 → https://aebonlee.github.io/rest04/
