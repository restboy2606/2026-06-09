## 2. 1차 개발 완료 항목

### 기반/디자인
- React 18 + Vite 5 + React Router 6(HashRouter) + Tailwind 3
- **5색 브랜드 팔레트**: navy(다크블루)·royal(로열블루) 주컬러 + sky(보조)·amber·mint(포인트)
- **다크/라이트 모드**: 라이트 기본, 토글 + localStorage + FOUC 방지(index.html 인라인)
- **모바일 최적화**: 반응형, 햄버거 패널, 가로 스크롤 탭
- rest03 디자인 이식: 컨테이너 1600px, 넉넉한 여백, 2단 다크 푸터

## 4. 개발일지 인덱스

1. `01` 초기 플랫폼 구축
2. `02` rest03 디자인 이식 + 실강의 연동
3. `03` 회사정보 반영 + 푸터 보강
4. `04` 웹 기초 + 파이썬 과정 추가
5. `05` 파이썬·R 연결 + 그리드 3×3
6. `06` (본 문서) 1차 개발 종료 종합 정리

## 5. 콘텐츠 운영 메모

- 영상 추가/교체: `src/data/site.js` 의 `videos[topic]` 에서 `youtubeId`(11자리)·`module` 수정
- 새 과정: `topics` 배열 + `videos` 객체에 같은 `key` 추가 → 탭/푸터/홈 자동 반영
- ⚠️ 유튜브는 **"미등록(링크 공개)"** 만 임베드 가능 (비공개 불가)
