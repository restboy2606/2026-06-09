## 2. 웹 개발 기초 (HTML 기초 수업)

- web01 ~ web20 을 모듈로 구분: HTML/CSS(web01~16) · JavaScript(js01,js02) · jQuery(jq01,jq02)
- **유튜브 링크 미수신** → `youtubeId` 비움(준비중 상태). 링크 받으면 `site.js` 에서 순서대로 채우면 됨.

## 3. 파이썬 프로그래밍 (소프트웨어 기초)

- 챕터 기반 모듈: 소개·설치 / 변수·터틀 / 변수·입출력 / 계산처리 / 자료형 / 반복문·함수
- 연결된 실제 영상(검증 완료):
  - `c41ut8-rlgs` — [경기대] Mac OS에 파이썬 설치하기
  - `tSzQHCAVibc` — 3장 반복학습 부분
- 나머지 강의는 링크 대기(준비중).

## 4. UX 보강

- `VideoCard` — `youtubeId` 가 없는 영상에 **"준비중" 배지**(앰버) 표시, 모달은 "영상이 연결되지 않았습니다" 안내.
- 주제 4개로 늘어도 헤더 드롭다운·탭·푸터·홈 카드가 `topics` 배열 기반으로 자동 반영.

## 5. 빌드/배포
